# Aviation Infrastructure Data Analysis

# Project Overview
This project explores aviation infrastructure data from Alaska using Python and Pandas.
The goal is to analyze airport facilities, ownership types, elevation patterns, and facility distribution.

# Dataset
The dataset contains multiple sheets:
- Facilities
- Runways
- Schedules
- Remarks

# The Facilities dataset includes information such as:

- Facility name
- City
- County
- Ownership
- Use type
- Elevation
- Facility type

# Project Steps
1 Data Cleaning

- Checked for missing values

- Filled missing values with "Unknown"

- Converted abbreviated ownership codes into readable labels

2 Data Overview

- Identified different aviation facility types

- Counted airports, heliports, and seaplane bases

- Measured the number of unique locations

3 Exploratory Data Analysis

- Calculated average facility elevation

- Identified highest and lowest elevation facilities

- Analyzed airport distribution by county

- Explored ownership and usage patterns

# Key Insights

- Airports are the most common aviation facility.

- Public ownership dominates aviation infrastructure.

- Certain counties contain significantly higher numbers of airports.

- Elevation varies significantly across facilities.
